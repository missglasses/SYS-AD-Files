1. Get Files

VirtualBox: Download from official site.

Win Server 2016 ISO: Get from Microsoft Evaluation Center.

2. Make VM

New VM -> Name: Windows Server 2016

Attach the ISO.

RAM: 2048 MB

HDD: 40 GB (Dynamic)

3. Install OS

Start VM.

Install Now -> Standard (Desktop Experience)

Custom install -> Select the drive -> Next.

Wait.

4. Set Password

At prompt, set a strong admin password (e.g., Server2016!).

Log in with it.

Done.
